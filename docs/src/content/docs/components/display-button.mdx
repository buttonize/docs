Renders an interactable button.

The button can be used to trigger a lambda, change pages within a Buttonize app, or interact with some other AWS services via Buttonize [Actions](/core-concepts/actions/#available-actions).

```ansi frame="none"
[38;5;39mDisplay.button[0m(props)
```

## Usage

<FrameworkContent>
   <Fragment slot="cdk">
    ```ts title="lib/MyStack.ts" collapse={1-5}
    import { Action, ButtonizeApp, Display } from '@buttonize/cdk'
    
    new ButtonizeApp(this, 'MyDemoApp', {
      name: 'Buttonize Docs Demo App'
    }).page('DemoPage', {
      body: [
        Display.button({
          label: 'Next Page',
          onClick: Action.aws.lambda.invoke(yourFunction)
        })
      ]
    })
    ```
   </Fragment>
</FrameworkContent>
