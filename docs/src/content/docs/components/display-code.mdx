Renders code within your Buttonize app with syntax highlighting.

```ansi frame="none"
[38;5;39mDisplay.code[0m(code, props)
```

## Usage

<FrameworkContent>
   <Fragment slot="cdk">
    ```ts title="lib/MyStack.ts" collapse={1-5}
    import { ButtonizeApp, Display } from '@buttonize/cdk'

    new ButtonizeApp(this, 'MyDemoApp', {
      name: 'Buttonize Docs Demo App'
    }).page('DemoPage', {
      body: [
        Display.code('const message = "Hello, World!"', {
          language: 'typescript',
          title: 'Code Example',
          frame: 'code'
        })
      ]
    })
    ```

   </Fragment>
</FrameworkContent>
