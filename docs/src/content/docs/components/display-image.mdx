Displays an image within your Buttonize app.

```ansi frame="none"
[38;5;39mDisplay.image[0m(url, props)
```

## Usage

<FrameworkContent>
   <Fragment slot="cdk">
      ```ts title="lib/MyStack.ts" collapse={1-5}
      import { ButtonizeApp, Display } from '@buttonize/cdk'

      new ButtonizeApp(this, 'MyDemoApp', {
        name: 'Buttonize Docs Demo App'
      }).page('DemoPage', {
        body: [
          Display.image('https://avatars.githubusercontent.com/u/72308571?s=50', {
            alt: 'Buttonize Log'
          })
        ]
      })
      ```

   </Fragment>
</FrameworkContent>
