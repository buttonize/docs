Displays a collapsible section with a label and body content.

```ansi frame="none"
[38;5;39mDisplay.section[0m(props)
```

## Usage

<FrameworkContent>
   <Fragment slot="cdk">
    ```ts title="lib/MyStack.ts" collapse={1-5}
    import { ButtonizeApp, Display } from '@buttonize/cdk'

    new ButtonizeApp(this, 'MyDemoApp', {
      name: 'Demo App'
    }).page('DemoPage', {
      body: [
        Display.section({
          label: 'Title',
          body: [
            Display.heading('Hello, world!')
          ],
          collapsed: true
        })
      ]
    })
    ```

   </Fragment>
</FrameworkContent>
