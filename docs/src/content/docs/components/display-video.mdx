Displays a video within your Buttonize app.

```ansi frame="none"
[38;5;39mDisplay.video[0m(url, props)
```

## Usage

<FrameworkContent>
   <Fragment slot="cdk">
    ```ts title="lib/MyStack.ts" collapse={1-5}
    import { ButtonizeApp, Display } from '@buttonize/cdk'

    new ButtonizeApp(this, 'DemoApp', {
      name: 'Demo App'
    }).page('DemoPage', {
      body: [
        Display.video('https://www.youtube.com/watch?v=giroZqjTJqE')
      ]
    })
    ```

   </Fragment>
</FrameworkContent>
