Displays an input field that can be used to interact with a chatbot.

```ansi frame="none"
[38;5;39mInput.chat[0m(props)
```

## Usage

<FrameworkContent>
   <Fragment slot="cdk">
    ```ts title="lib/MyStack.ts" collapse={1-5}
    import { ButtonizeApp, Input } from '@buttonize/cdk'

    new ButtonizeApp(this, 'DemoApp', {
      name: 'Demo App'
    }).page('DemoPage', {
      body: [
        Input.chat({
          id: 'chatInput',
          onMessage: {
            type: 'sendMessage',
            content: 'Hello, World!'
          },
          placeholder: 'Type your message...',
        })
      ]
    })
    ```

   </Fragment>
</FrameworkContent>
