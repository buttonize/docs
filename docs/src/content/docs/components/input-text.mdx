## Runtime Storage

The value in the runtime storage is string value of the text input.

```ts title="lib/MyStack.ts" "'name'"
Input.text({
	id: 'name',
	label: 'What is your name?'
})
```

```json title="Runtime storage" frame="terminal" ""name""
{
	"name": "David Copperfield"
}
```
