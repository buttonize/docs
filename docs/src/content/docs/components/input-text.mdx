## Usage

## Runtime State

The value in the runtime state is string value of the text input.

```ts title="lib/MyStack.ts" "'name'"
Input.text({
	id: 'name',
	label: 'What is your name?'
})
```

```json title="Runtime state" frame="terminal" ""name""
{
	"name": "David Copperfield"
}
```
