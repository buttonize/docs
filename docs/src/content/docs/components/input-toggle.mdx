Displays an interactable toggle button within a Buttonize page.

```ansi frame="none"
[38;5;39mInput.toggle[0m(props)
```

## Usage

<FrameworkContent>
   <Fragment slot="cdk">
    ```ts title="lib/MyStack.ts" collapse={1-5}
    import { ButtonizeApp, Input } from '@buttonize/cdk'

    new ButtonizeApp(this, 'DemoApp', {
      name: 'Demo App'
    }).page('DemoPage', {
      body: [
        Input.toggle({
          id: 'toggleSwitch', // the name of the variable in the runtime state
          label: 'Enable Feature',
        }),
        Display.text("toggleSwitch's value: {{toggleSwitch}}"),
      ]
    })
    ```

   </Fragment>
</FrameworkContent>

## Runtime State

The value in the runtime state is the boolean value of the toggle input.

```ts title="lib/MyStack.ts" "'name'"
Input.toggle({
	id: 'bool',
	label: 'My Checkbox'
})
```

```json title="Runtime state" frame="terminal" ""name""
{
	"bool": true
}
```

As demonstrated by the Usage example above, you can access variables from the Runtime State
via the "\{\{bool\}\}" syntax from elsewhere in your Buttonize app.
