Displays an interactable checkbox within a Buttonize page.

```ansi frame="none"
[38;5;39mInput.toggle[0m(props)
```

## Usage

<FrameworkContent>
   <Fragment slot="cdk">
    ```ts title="lib/MyStack.ts" collapse={1-5}
    import { ButtonizeApp, Input } from '@buttonize/cdk'

    new ButtonizeApp(this, 'DemoApp', {
      name: 'Demo App'
    }).page('DemoPage', {
      body: [
        Input.toggle({
          id: 'toggleSwitch',
          label: 'Enable Feature',
        })
      ]
    })
    ```

   </Fragment>
</FrameworkContent>
